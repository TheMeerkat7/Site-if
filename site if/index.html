Quer ajuda para traduzir seus códigos python?

Nossa empresa tem a solução!

Traduzimos seu código python para outras linguagens usando nossa IA!

Venha conhecer nossa tecnologia!

Como funciona?

  1. Analisamos o código recebido, para certificar que não há nenhum erro.
  2. Indicamos para a IA a linguagem para tradução e colocamos o código.
  3. A tradução é feita e enviada para você!

Exemplo:

Código em python:

salario = int(input("Digite seu salario:"))
if salario < 1400 :
a = salario * 0.6
b = salario * 0.2
c = salario * 0.2
elif salario >=1400 and salario < 3600:
a = salario * 0.5
b = salario * 0.3
c = salario * 0.2
elif salario >=3600 and salario < 6200:
a = salario * 0.4
b = salario * 0.3
c = salario * 0.3
elif salario >=6200:
a = salario * 0.3
b = salario * 0.4
c = salario * 0.3
print("\n A divisao do seu salario e de: ",a," para gastos essenciais, ", b, "para lazer e ", c, " para investir")
